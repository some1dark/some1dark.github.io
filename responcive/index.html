Womit werden HTML-Seite erstellt?

Im Grunde kann man Webseiten in jeden Texteditor erstelln. Hierbei ist zu beachten das die datei mit der Endung *.txt nicht als Webseite anerkannt wird. Sobald wir die Datei als *.html (* steht für die Dateinamen) speichern kann der Browser diese dan richtig auslesen und darstellen sobald der Inhalt dementsprechen in HTML vormartiert ist.

Bei den unterschiedlichen Texteditoren gibt es wesentliche Unterschiede:
Einige sind mit erweiterten Funktionen ausgestettet um das coden einer HTML-Seite zu vereinfachen. Dabei ist die meist genuzte Erweierungsfunktion das so genante Syntax-Highlighting und wird je nach Programmiersprache einen anderen Syntax hervorheben und Code Compeletion

Adobe Brackets

Haupsächlich von Adobe für das Entwickeln von Webprojekten gedacht, ist es ein leistungsstarkes Open-Source Code-Editor das selbst mit HTML, CSS & JavaScript entwickelt wurde. Da dieses Tool selbst aus Web-Technologien entstand, ist es wegen seiner OpenSource MIT Lizenz selbst erweiterbar.
Adobe Brackets ist zum Bearbeiten von HTML, CSS & JavaScript mit sehr inovativen Funktionen ausgestattet. Er verfügt über sehr gute Syntax-Highlighting die nach ermessen bearbeitet werden kann. Eine Live-Ansicht wird auch geboten, allerdings nur für den Google Chrome Browser. Doch die Live-Ansicht bietet noch viel mehr, Brackets hebt dynamisch die Elemente im Browser optisch hervor, andem im Quellcode bearbeitende stelle.



Live-Ansicht

Brackets bietet bietet einem eine Live-Ansicht die derzeit nur in Google Chrome verfügbar ist. Weite Browseruntestüzung ist für die Zukunft geplant. Das besondere an der Live-Ansicht ist das Brackets dynamisch die Elemente in Chrome hervorhebt wo sich der Browser in Quellcode befindet. Dieses gild für die HTML undd CSS Datei.



Quick Edit

In der Regel bestehen Webprojekte aus mehreren Datein die miteinander Verknüpft sind. In Brackets kann man in der HTML verknüpfte CSS und JavaScript Datein mit Hilfe des Inline-Editors bearbeiten ohne direkt in die dazu geörende Datei zu wechseln. Um ein Element direkt zu bearbeiten einfach selektieren und Strg-E (Windows) und CMD-E (Mac) drücken.



Weitere Features

Neben Syntax-Highlighting und Code-Completion ist Brackets auf Web-Technologien basiert und kann vom User individuel angepasst werden. Hiefür sind bereits viele Plug-ins frei verfügbar. Empfelende Erweiterungen wären:

  • W3C Validation
  • Code Folding
  • Brackets Working File Tab
  • Emmit
  • Brackets Icon
  • LESS AutoCompile

Um Brackets mit weiteren Features auszustatten müsst ihr im oberen Rechten Teil des Fensters auf den Baustein clicken. Dort könnt ihr nach Erweiterungen suchen und installieren.



WYSIWYG-Editoren

WYSIWYG Editoren sind graphisch orientierte Editoren die es erlauben seine HTML-Seiten wie in einem Layout- oder Satzprogramm auf visueller Basis zu gestalten. Ich werde nicht weiter auf diese Editoren eingehen da sie meines erachtens nach nicht geeignet sind um Webdesign zu lernen obwohl sie einem ein gutes Verständnis dafür geben können wie eine Website aufgebaut ist.


Bestandteile der HTML-Datein

Jede Webseite ist im grunde eine HTML-Datei. Die HTML-Datei ist eine Text-Datei mit der Dateiendung *.html oder *.htm und ist befüllt mit HTML-Tags & Inhalt die dem Brwoser angeben wie der Inhalt wiederzugeben ist. Anders gesagt, HTML sind in einem Format angelegt um den Browser Anleitungen zu geben wie er alles anzeigen soll.


Tags

Die Primäre Aufgabe des Browsers ist es HTML Dokumente darzustellen. Unerheblich ist es dabei ob sich die HTML-Datein lokal auf dem eigenen Rechner oder auf einem Server um die habe Welt liegen. Die HTML-Datein enthalten Informations Steuerbefehle (Auszeichnungen), sogenante Tags um dem Browser zu sagen wie die Seite auszusehen hat.

Der Quelltext einer HTML-Datei kann in zwei Kategorien unterteilt werden:

  • Inhalt
    Normaler Text, der Information zu Seite enthält.
  • Markup
    Der Code, der beschreibt wie der Inhalt der Seite zu strukturieren ist.
